Modelo de Guimarães (2009) — DNIT 179/2018-IE seção 10.1
Expressão do modelo
εp(%) = ψ₁ · (σ₃ / p₀)^ψ₂ · (σd / p₀)^ψ₃ · N^ψ₄
p₀ = 101,325 kPa (pressão atmosférica de referência)
Como os parâmetros foram obtidos
1O modelo Random Forest previu os valores de εp para os 13 pontos de leitura da curva (N = 1, 10, 50... 150.000 ciclos)
2Usando σ₃, σd e N como variáveis independentes, aplicou-se regressão não linear múltipla (método dos mínimos quadrados) para ajustar os 4 parâmetros ψ₁..ψ₄ à curva prevista
3O ajuste minimiza a diferença entre a curva do Random Forest e a curva analítica do modelo de Guimarães
